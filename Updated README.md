# flight-booking-app-webflux
1) Built using Java Spring boot, MongoDB
2) Written in Reactive style of programming using Webflux
3) Code is Non-blocking and asynchronous in nature
4) The app contains 3 Post, 2 Get and 1 Delete endpoint
5) POST - Add flight, Search flights
6) GET - Book Ticket, View Ticket, View history of bookings
7) DELETE - Cancel Ticket
8) Implemented Sonar Cloud Quality analysis and implemented its suggestions
9) Used Jmeter for Stress testing the APIs
10) Sonar cloud analysis link - https://sonarcloud.io/project/overview?id=Rakesh8086_flight-booking-app-webflux
